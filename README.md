 StudentSystemWeb2

Полнофункциональная веб-система для управления студентами, оценками, предметами и учебной аналитикой. Проект разработан на основе Spring Boot, Thymeleaf и PostgreSQL, и предназначен для учебных заведений, преподавателей и студентов.

Система включает удобную панель администратора, журнал оценок, личные кабинеты пользователей и визуальную аналитику успеваемости.

 Основные возможности:
 Функционал для преподавателей

Просмотр списка студентов

Выставление оценок

Просмотр индивидуального журнала успеваемости

Просмотр статистики и аналитики по оценкам

Удобная навигация между предметами и группами

 Функционал для студентов:

Личный кабинет

Просмотр текущих оценок

История успеваемости по каждому предмету

Прозрачная и понятная система отображения журнала

Функционал для администратора:

Управление студентами

Управление преподавателями

Управление предметами и группами

Панель администратора с ключевыми показателями

Просмотр системной аналитики
