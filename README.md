StudentSystemWeb2 — это полнофункционционная веб-система, предназначенная для управления студентами, учётной информацией, оценками и учебной аналитикой. Проект разработан на базе технологий Spring Boot, Thymeleaf и PostgreSQL, и ориентирован на использование в учебных организациях, преподавателями и самими студентами.

Система предоставляет удобную панель администратора, цифровой журнал успеваемости, личные кабинеты пользователей и модуль визуальной аналитики, обеспечивающий прозрачность и наглядность учебного процесса.
